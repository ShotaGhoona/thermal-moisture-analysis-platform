# プロジェクトビジョン: 熱水分移動解析Webアプリケーション

## プロジェクト概要

**「建築材料の熱・水分移動をブラウザで簡単にシミュレーションできるWebアプリケーション」**

現在、Julia言語で動作している熱水分同時移動解析標準プログラムをWebブラウザ上で誰でも簡単に使えるようにします。

## 現状の課題

### 専門性の壁
- **Julia言語の知識が必要** → 建築の専門家でもプログラミングスキルが必要
- **複雑なCSVファイル編集** → パラメータ変更にテキストエディタとファイル知識が必要
- **コマンドライン操作** → GUI不慣れなユーザーには敷居が高い

### 利用環境の制約
- **開発環境構築が必要** → Julia + 各種パッケージのインストール
- **ローカル実行のみ** → 計算結果の共有や比較が困難
- **可視化機能が限定** → グラフ作成に別途プログラミングが必要

## 解決策: Webアプリケーション化

### フロントエンド（Next.js）
**「誰でも直感的に操作できるUI」**

- **スライダー・ドロップダウン** でパラメータを視覚的に調整
- **リアルタイムプレビュー** で設定変更の影響を即座に確認
- **インタラクティブグラフ** で計算結果を美しく可視化
- **レスポンシブデザイン** でスマホ・タブレットからも利用可能

### バックエンド（Julia）
**「既存の高精度計算エンジンをそのまま活用」**

- **既存Juliaコードの再利用** で開発コストを最小化
- **API化** でフロントエンドとの疎結合を実現
- **並列計算対応** で複数ユーザーの同時利用をサポート
- **計算進捗の可視化** でユーザー体験を向上

## 段階的開発計画

### Phase 1: MVP（最小実用版）
**目標: 1次元計算をWebブラウザで実行**

- 基本パラメータ（室温・外気温・壁材料）の設定
- 簡単な温度・湿度変化グラフの表示
- 計算結果のCSVダウンロード

### Phase 2: 機能拡張版
**目標: 実用的な解析ツールとしての完成**

- 3次元計算・ネットワーク計算への対応
- 複数ケースの比較機能
- 詳細な可視化（ヒートマップ・アニメーション）

### Phase 3: プラットフォーム版
**目標: 建築業界標準ツールとしての確立**

- ユーザー管理・プロジェクト保存機能
- API公開による他システムとの連携
- クラウド計算による大規模解析対応
