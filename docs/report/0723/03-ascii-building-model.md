## データ構造の対応関係

### room_condition.CSV → 空間定義
```
num │ name              │ vol(m³) │ 役割
────┼───────────────────┼─────────┼─────────────────────────
1   │ outdoor           │ 1000.0  │ 外界気象・境界条件
2   │ storage room      │ 646.2   │ メイン収蔵空間
3   │ special storage   │ 112.1   │ 特別収蔵空間  
4   │ machine room      │ 64.1    │ 機械・設備室
5   │ vestibule         │ 57.9    │ 前室・緩衝空間
6   │ doublewallA       │ 67.1    │ 二重壁中空層A
7   │ doublewallB       │ 100.5   │ 二重壁中空層B
8   │ attic             │ 45.8    │ 屋根裏緩衝空間
9   │ underfloor        │ 166.9   │ 床下緩衝空間
```

### wall_condition.CSV → 熱水分移動経路（26壁体）
```
壁体タイプ    │ 接続関係              │ 材料構成           │ 役割
─────────────┼───────────────────────┼───────────────────┼─────────────────
wall1~5      │ outdoor ←→ 各室       │ 断熱・防湿材      │ 外壁・断熱
ceiling1~2   │ 各室 ←→ attic        │ 断熱・気密材      │ 天井・断熱  
floor1~3     │ 各室 ←→ underfloor   │ 断熱・防湿材      │ 床・断熱
roof1        │ attic ←→ outdoor     │ 防水・断熱材      │ 屋根・防水
ground       │ underfloor ←→ 地盤   │ 防湿・断熱材      │ 基礎・防湿
stored_item  │ storage内収蔵物       │ 木材・紙等        │ 調湿・熱容量
```

### opening_condition.CSV → 換気経路（15開口部）
```
開口ID │ 接続関係          │ Type     │ 換気量(m³/s) │ 役割
───────┼───────────────────┼──────────┼──────────────┼─────────────────
1~5    │ outdoor ←→ 各室   │ constant │ 0.008~0.020 │ 外気導入
6~10   │ 各室間            │ constant │ 0.000~0.020 │ 室間換気
11~15  │ 上下空間          │ constant │ 0.37~1.67   │ 自然換気
```
